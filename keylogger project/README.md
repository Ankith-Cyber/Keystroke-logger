\# Keyboard Event Logger (Educational Project)



\## Description

This project demonstrates how keyboard input events can be captured for learning

purposes such as cybersecurity awareness and accessibility testing.



\## Purpose

This project is created strictly for educational use.



\## Features

\- Captures keyboard input events

\- Displays logged keys for learning

\- Simple and beginner-friendly
